1. If you were to describe semantic HTML to the next cohort of students, what would you say?

Semantic html means the elements in an html file or website. It is the language use to communicate
what you want the page to do.

2. Name two big differences between display: block; and display: inline;.

Display inline means it's displayed in a line (on the same line) vs block being displayed in seperate blocks of space. Ex, span is inline 
and will show up on the same line. Div or even p creates a new line or block.
